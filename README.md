# My Neovim Config

## Requirements
- WSL2 Ubuntu
- Neovim 0.11+
- Node.js
- Git

## Install
curl -s https://raw.githubusercontent.com/ayyogz/dotfiles/main/install.sh | bash

## Languages supported
- C / C++
- Java
- Python
- JavaScript
- HTML / CSS
- SQL

## Key bindings
- F5 — save and run
- Space+e — file explorer
- Ctrl+\ — toggle terminal
- Space+ls — live server
